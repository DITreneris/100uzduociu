# Greitas pradžios gidas

## Naudojimas

### 1. Atidarykite svetainę
Atidarykite `index.html` naršyklėje arba per GitHub Pages.

### 2. Raskite užduotį
- Naršykite pagal skyrių
- Naudokite paiešką: įveskite raktažodį (pvz., "el. laiškas", "kaina", "PVM")

### 3. Kopijuokite užduotį
1. Spustelėkite "Kopijuoti" mygtuką
2. Tekstas bus nukopijuotas į iškarpinę

### 4. Naudokite AI įrankyje
1. Atidarykite ChatGPT, Claude, Copilot ar kitą AI įrankį
2. Įklijuokite nukopijuotą tekstą
3. Pakeiskite [laukus] savo duomenimis:
   - `[produktas]` → jūsų produkto pavadinimas
   - `[klientas]` → kliento vardas
   - `[kaina]` → konkreti kaina

### 5. Gaukite rezultatą
AI sugeneruos rezultatą. Jei reikia, papildykite užklausą.

## Pavyzdys

**1. Pasirinkite užduotį:**
```
Pardavimai → El. laiškas klientui
```

**2. Kopijuokite:**
```
Parašyk aiškų, draugišką el. laišką klientui, 
besidominčiam [produktas]. Paaiškink pagrindinius 
privalumus ir pasiūlyk tinkamų alternatyvų.
```

**3. Pakeiskite laukus:**
```
Parašyk aiškų, draugišką el. laišką klientui, 
besidominčiam Arduino Uno R3. Paaiškink pagrindinius 
privalumus ir pasiūlyk tinkamų alternatyvų.
```

**4. Įklijuokite į ChatGPT ir gaukite rezultatą.**

## Patarimai

- Visada pakeiskite [laukus] - tai padės gauti tikslesnį rezultatą
- Eksperimentuokite su skirtingais AI įrankiais
- Išsaugokite gerai veikiančias užduotis AI įrankyje
